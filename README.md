# Theories of Consciousness

## Neuroscientific, Philosophical, Computational, and Physical Perspectives

This repository contains a `bookdown` project for an academic book exploring and critically comparing major theories of consciousness across neuroscience, philosophy, cognitive science, artificial intelligence, and physics.

The project aims to provide a comprehensive and interdisciplinary overview of consciousness studies by integrating scientific, philosophical, and computational perspectives into a single structured resource.

---

## About the Book

Consciousness remains one of the most profound and unresolved questions in science and philosophy.

This book examines how different disciplines attempt to explain:

- Subjective experience
- Self-awareness
- Perception
- Cognition
- Qualia
- Agency
- Conscious processing
- The relationship between mind and matter

The book critically evaluates competing theories, discusses their empirical support, and highlights unresolved conceptual and scientific challenges.

---

## Topics Covered

The repository currently includes chapters on:

- Dualism
- Materialism and Physicalism
- Functionalism
- Emergentism
- Global Workspace Theory
- Integrated Information Theory (IIT)
- Higher-Order Thought Theory
- Predictive Processing and Active Inference
- Recurrent Processing Theory
- Attention Schema Theory
- Computationalism
- Bayesian Brain Approaches
- Panpsychism
- Quantum Theories of Consciousness
- Illusionism
- Embodied and Enactive Cognition
- Disorders of Consciousness and Altered States
- Artificial Intelligence and Machine Consciousness

Additional chapters and revisions are planned as the project evolves.

## Building the Book

### Prerequisites

Install the required R packages:

```r
install.packages("bookdown")
install.packages("rmarkdown")
```

---

### Render the Book

To build the book locally:

```r
bookdown::render_book("index.Rmd")
```

The rendered HTML output will be generated in:

```text
_book/
```
