# Figures

This folder contains placeholder diagrams in Mermaid format.

Suggested figures to develop:

1. `global-workspace-theory.mmd`  
   A schematic showing unconscious processors, attention, global broadcasting, memory, language, and action systems.

2. `predictive-processing-hierarchy.mmd`  
   A hierarchical prediction-error diagram.

3. `theory-map.mmd`  
   A conceptual map of major theory families.

You can render Mermaid diagrams using VS Code extensions, GitHub Markdown preview, or online Mermaid tools, then save them as SVG or PNG for inclusion in the book.

Example Bookdown syntax:

```markdown
![Global Workspace Theory architecture](figures/gwt-diagram.png)
```
